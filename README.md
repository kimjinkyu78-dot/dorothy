# 코인 시세 대시보드

업비트(국내)와 바이낸스(해외)의 메이저 5대 코인(BTC, ETH, SOL, XRP, DOGE) 시세를 한 화면에서 비교하고, 물타기(평균단가) 계산기를 제공하는 웹 앱입니다.

## 주요 기능

- **실시간 시세**: 업비트(KRW) · 바이낸스(USDT) 현재가 및 24h 변동률
- **김치프리미엄**: 업비트 KRW vs 바이낸스 USDT→KRW 환산가 괴리율(%)
- **실시간 청산**: 바이낸스 선물 실제 청산 체결 · WebSocket · 표 형식 (내역/구간별 집계)
- **추정 청산맵 (OI)**: 미결제약정 + 롱/숏 비율 + 레버리지 모델 · 청산 밀집 구간 추정
- **과거 OHLC**: 24시간 / 7일 / 30일 구간 고가·저가·종가
- **물타기 계산기**: 바이낸스 USDT-M 선물 기준 · 롱/숏 · 레버리지 · 청산가·ROE 계산
- **갱신 방식**: WebSocket + 8초 REST 폴링 (API rate limit 준수)
- **연결 상태**: 거래소별 정상/지연/오류 표시

## 사용 API (공개, API 키 불필요)

| 거래소 | 용도 | 엔드포인트 |
|--------|------|------------|
| 업비트 | 현재가 | `GET /v1/ticker` |
| 업비트 | OHLC | `GET /v1/candles/minutes/60`, `/v1/candles/days` |
| 업비트 | USDT/KRW 환율 | `GET /v1/ticker?markets=KRW-USDT` |
| 업비트 | 실시간 | `wss://api.upbit.com/websocket/v1` |
| 바이낸스 | 현재가 | `GET /api/v3/ticker/24hr` |
| 바이낸스 | OHLC | `GET /api/v3/klines` |
| 바이낸스 | 실시간 | `wss://stream.binance.com:9443/stream` |

## 주의사항

- 업비트·바이낸스 API는 **rate limit** 및 **점검 시간**이 있을 수 있습니다.
- WebSocket 연결 실패 시 REST 폴링으로 자동 보완하지만, 네트워크 환경에 따라 지연될 수 있습니다.
- USDT/KRW 환율은 업비트 `KRW-USDT` 마켓 기준입니다.
- **본 앱은 정보 제공 및 계산 목적**이며, 실제 주문·자동매매 기능은 포함하지 않습니다.
- **투자 판단과 손실에 대한 책임은 전적으로 사용자에게 있습니다.**

## 선물 물타기 계산 공식

```
추가 포지션 수량 = (추가 증거금 × 레버리지) / 진입가 × (1 - 수수료율)
새 평균 진입가 = (기존수량×기존진입가 + 추가수량×추가진입가) / 총수량
총 증거금 = 총 포지션 가치 / 레버리지

롱 미실현손익 = (현재가 - 평균진입가) × 총수량
숏 미실현손익 = (평균진입가 - 현재가) × 총수량
ROE = 미실현손익 / 총증거금 × 100

롱 청산가 ≈ 평균진입가 × (1 + 유지증거금률 - 1/레버리지)
숏 청산가 ≈ 평균진입가 × (1 + 1/레버리지 - 유지증거금률)
```

- 기본 수수료: 바이낸스 0.1% (내부 반영)
- 유지증거금률: 0.5% 가정 (거래소·코인별 상이)
